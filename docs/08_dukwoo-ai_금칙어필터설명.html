3
containsBanned(keyword, allowList) 필터 적용
// v7: allowList 추가
for (const m of BANNED_MODELS) {
  if (lower.includes(m) && !allowList.includes(m)) return true;
} // ← "토레스"가 allowList에 있으므로 통과!
"토레스" 포함 키워드 → allowList에 "토레스" 있음 → 통과 ✓
"소나타" 포함 키워드 → allowList에 없음 → 삭제 ✗ (다른 차종 혼입 차단)
↓
✅ 200 OK — 키워드 42개 → Naver 검증 → 최종 결과 반환

핵심 요약

🔴 v6 (수정 전)
상품명에 차종이 있으면
→ Gemini가 생성한 키워드도 당연히 해당 차종 포함
→ containsBanned()가 전부 삭제
→ 빈 배열 → 502 에러

실패 상품: 토레스, 그랜저, 쏘렌토, SM6, EV6, 카니발, 스포티지, 소나타...
🟢 v7 (수정 후)
상품명에서 차종 자동 추출 → allowList
→ 해당 차종 키워드는 필터 통과
→ 다른 차종은 여전히 차단
→ 정상 결과 반환
