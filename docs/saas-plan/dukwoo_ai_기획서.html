2 시장 진단 (요약)

검증 완료 — 경쟁사·시장 도구 검색·비교 끝
경쟁군핵심우리 위치
키워드 SaaS (셀러레이더·아이템스카우트·판다랭크 등)1인 셀러용. 1건씩 처리. 평생요금제 ~ 월 ₩1만정면 경쟁 회피 시장 다름
통합관리 솔루션 (사방넷 80% / 샵링커 / 이지어드민 / UFlat)주문·재고·등록 통합. 키워드 기능 부재 또는 약함 (샵링커만 월 100건 AI)애드온 침투 우리 진짜 잠재 시장
위탁대행 시장사람이 1상품 80원 수작업. 5~7일 대기대체 ₩55만/1만 + 10분
핵심 발견: 시장에 "엑셀 일괄 처리 SaaS" 카테고리 자체가 없음. 우리만 가능. 통합관리 솔루션 사용자 5.6~12.6만명이 잠재 고객.

3 타겟 / 페르소나

🎯 메인 타겟 — 위탁대행사 사장

  • 자기 셀러 50~100명 관리
  • 매월 신상 1만~10만 상품 등록 필요
  • 현재 사람 손으로 처리 → 시간·인건비 부담
  • ARPU: 월 ₩55만~₩500만
  • 한국 200~500곳

🎯 서브 타겟 — 종합 도매상·무역회사

  • 덕우·OEM 수입 회사들
  • 월 500~5,000 신상 등록
  • UFlat·사방넷 등 통합 솔루션 사용 중
  • ARPU: 월 ₩6만~₩55만
  • 한국 수천 곳

🎯 서브 타겟 — 중대형 셀러

  • 상품 1,000~10,000개 보유
  • 안 이사 같은 사람
  • 매월 일부 리프레시 + 신상 처리
  • ARPU: 월 ₩6만~₩55만
  • 한국 5,000~10,000명

❌ 비타겟

  • 1인 셀러 (상품 10~100개) — 셀러레이더로 충분
  • 신규 셀러 — 우리 ROI 정당화 어려움
  • 비통합관리 사용자 — 엑셀 안 다룸

9 빌드 로드맵 (6주)

주차Phase작업완료 기준
0주차 (이번 주)Phase 0 — 기획·셋업도메인·결제사 가입·DB 스키마·디자인 톤도메인 활성화, 토스 가맹점 신청, SQL 마이그레이션 실행
1주차Phase 1 — 인증Supabase Auth·회원가입·로그인·프로필실제 가입 → 잔액 10 부여
2~3주차Phase 2 — 결제·크레딧토스페이먼츠 연동·패키지 결제·크레딧 차감실결제 → 잔액 증가 → 사용 시 차감
3~4주차Phase 3 — 엑셀 양식 호환사방넷·샵링커 양식 신규 지원, UFlat 유지3종 양식 모두 정상 처리
4주차Phase 4 — 대시보드잔액·사용 이력·다운로드·결제 이력사용자 본인 정보 모두 조회 가능
5주차Phase 5 — 관리자회장님 어드민 패널·환불·강제 충전회장님 본인 운영 가능
5주차Phase 6 — 법무이용약관·개인정보·환불정책·사업자정보변호사 검토 통과
5~6주차Phase 7 — 마케팅랜딩·가격·FAQ·ROI 계산기·블로그 5편SEO 트래픽 시작 + 가입 전환 흐름 완성
6주차Phase 8 — QA·베타안 이사 + 위탁대행사 3~5곳 베타실사용 피드백 수렴

11 리스크 & 대응

리스크영향대응
네이버 API 약관 — 상업적 재배포 가능 여부🔴 사업 모델 흔들림약관 사전 확인. 막히면 KQS 점수만 자체 산출 + 검색량 비공개 처리
샵링커 키워드 기능 확장🟡 차별성 약화대량(1만+) 처리는 우리만. 양식 호환·가격으로 차별
사방넷·이지어드민이 자체 키워드 도구 도입🟡 1~2년 후 가능성그 전에 시장 점유 + 위탁대행사 화이트라벨 lock-in
고객 획득 비용 (CAC) 높음🟡 초반 적자위탁대행사 B2B 영업 = CAC 낮음. 셀러 카페 광고 병행
토스페이먼츠 가맹 심사 거절🟢 결제 지연나이스페이·KG이니시스 대안
이탈률 높음 (Churn)🟡 충전식이라 자연스럽지만 재구매율 낮으면 문제월간 사용 리포트 자동 발송·잔액 부족 안내·재구매 할인 이벤트

12 회장님 승인 필요 사항 / 다음 액션

먼저 결정해주실 6가지:
  1. 운영 법인 — 덕우무역(한국) / 브라이튼 PTE(SG) / 신규?
  2. 도메인 dukwoo.ai — 이미 보유? 아니면 즉시 구매
  3. 로고·브랜드 톤 — 기존 keyword-tool 디자인 재활용 / 새로 디자인
  4. 가격 패키지 — 위 4단계 그대로 OK?
  5. 출시 시점 목표 — 6주 후(7/13 베타)? 또는 더 빠르게 / 천천히?
  6. 베타 모집 채널 — 안 이사 위탁 인맥 / 셀러 카페 / 광고?
승인 후 즉시 진행할 작업 (이번 주):
  1. 회장님: 도메인 확인·구매 + 토스페이먼츠 신청 + 법인 정보 정리
  2. 엠버: DB 스키마 SQL 작성 + Supabase 적용 + Next.js 프로젝트 생성 + Vercel 연결
  3. 엠버: 사방넷·샵링커 양식 컬럼 구조 분석 (샘플 엑셀 확보 필요 — 회장님이 1~2개 구해주시면)
  4. 엠버: 메모리에 본 기획서 박제 (projects/dukwoo_ai_saas.md 신규)
