✅ 우리 차별점 (확실)

  • 엑셀 in/out 완전 자동 — 시장에 우리만
  • UFlat 양식 그대로 지원 — 묶음코드 · 마이카테 · 이미지 URL 다 인식
  • 1만 상품 5~10분 — 사람 167시간 → 봇 10분
  • KQS 점수 + Gemini 카테고리 매핑 — 단순 검색량 X, 검증된 점수
  • generate-title — 키워드 기반 SEO 상품명 자동 생성

⚠️ 약점·리스크

  • 1인 셀러용 UI 부재 — 검색창·즉시 분석 같은 가벼운 기능 없음
  • 네이버 API 약관 — 상업적 재배포 가능 여부 확인 필요
  • 인지도 0 — 셀러레이더는 이미 시장 점유
  • 네이버 정책 변경 리스크 — 네이버가 검색량 API 막으면 모두 위험

차별 축:
1. 처리 단위가 다름 — 1개씩 vs 1만 개 일괄
2. 타겟이 다름 — 1인 셀러 vs 대량 운영자
3. 가격 정당화 다름 — 도구비 vs 인건비 절감 ROI
4. 네이버 데이터 모자라면 → 덕우 무역망 OEM 소싱 풀스택 (v2)

왜 경쟁 안 함:
셀러레이더가 우리한테 못 따라옴 = "1만 개 한 번에" 같은 UX는 일반 셀러용 SaaS 구조가 완전 다름. 그쪽 가입자는 상품 10~100개 1인 셀러라서 일괄 처리 수요 자체가 없음.
