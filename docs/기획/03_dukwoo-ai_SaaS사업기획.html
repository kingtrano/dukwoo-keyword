Why Now

셀러의 키워드 지옥

네이버 검색 알고리즘은 키워드가 곧 매출이다. 그런데 87%의 셀러가 아직도 수작업으로 키워드를 찾고 있다.

😤

Pain 1: 시간 낭비

상품 하나당 키워드 리서치에 30분~1시간. 100개 상품이면 일주일. 엑셀에 붙여넣고, 검색량 하나하나 확인하고, 다시 정리하고.

🎯

Pain 2: 감에 의존

경쟁도, CTR, 광고밀도 같은 데이터 없이 "많이 검색되겠지" 하고 찍는다. 검색량 높은데 경쟁 치열한 키워드에 매달린다.

📉

Pain 3: 갱신 안 됨

키워드 트렌드는 매월 바뀌는데, 한 번 세팅하고 방치. 시즌이 지나도 "겨울용"이 달려있고, 트렌드 키워드는 놓친다.

Product

핵심 기능

덕우무역에서 실전 검증된 3-Step 파이프라인을 SaaS로 패키징. AI가 키워드를 찾고, 데이터로 등급을 매기고, 매월 자동으로 갱신한다.

MVP
🤖

AI 키워드 확장

상품명 하나 입력하면 Gemini AI가 7개 카테고리(정식명칭, 기능, 구어체, 소재, 시즌 등)에서 50~70개 키워드를 즉시 생성. 상품 이미지 있으면 Vision AI가 불일치 키워드 자동 제거.

MVP
📊

KQS 등급 시스템

네이버 API 6개 필드(검색량, PC/모바일, 경쟁도, CTR, 광고밀도)를 한 번에 수집. KQS 공식으로 S/A/B/C/D 등급 자동 산출. "핵심공략" "적극활용" 전략 태그 자동 부여.

MVP
⚡

즉시 캐시

한 번 검색된 키워드는 DB 캐시. 두 번째부터 0.3초 즉시 응답. 캐시 적중률이 높아질수록 API 비용 절감 + 사용자 경험 향상.

V2
📦

엑셀 일괄 처리

스마트스토어 상품 엑셀 통째로 업로드. 묶음코드별 자동 그룹핑, 고트래픽↔롱테일 페어링 분배. 다운로드 한 번이면 전체 상품 키워드 세팅 끝.

V2
🔄

월간 자동 갱신

네이버 검색량 업데이트 주기(매월 7일)에 맞춰 보유 키워드 전체 자동 갱신. 트렌드 변화 알림. 시즌 키워드 자동 추천.

V3
🏪

스토어 모니터링

내 스토어 + 경쟁 스토어 키워드 추적. 순위 변동 알림. 경쟁사가 새로 올린 키워드 감지. 카테고리별 트렌드 리포트.

쿠팡/11번가 키워드 지원 + 자동 상품명 생성 AI + SEO 점수 진단 + 광고 자동 입찰 연동. 한국 이커머스 SEO 원스톱 솔루션.
Moat

경쟁 해자

기존 키워드 도구(아이템스카우트, 셀러마스터)와의 차별점.

🤖

AI 자동 확장 vs 수동 입력

기존 도구: 셀러가 키워드를 직접 입력해서 검색량만 확인. KeywordForge: 상품명만 넣으면 AI가 7개 카테고리에서 자동 발굴. 셀러가 생각 못한 키워드까지 찾아냄.

📊

KQS 등급 vs 검색량만

기존 도구: 검색량 숫자만 보여줌. KeywordForge: 경쟁도×CTR×광고밀도를 종합한 KQS로 "이 키워드가 실제로 좋은가"를 판단. 전략 태그까지.

🖼️

이미지 Vision 검증

세계 유일. 상품 이미지를 Vision AI로 분석해서 상품과 맞지 않는 키워드를 자동 제거. "자동차커버"인데 "오토바이커버" 키워드가 빠지는 정밀도.

🔄

데이터 축적 = 네트워크 효과

유저가 많아질수록 캐시 적중률 ↑, 키워드 DB ↑, 트렌드 정확도 ↑. 후발주자가 따라오기 어려운 데이터 해자. 12개월이면 100만+ 키워드 축적.
